# MagicPage — RadioPlay Tonelada Elétrica

Rádio online da banda Tonelada Elétrica, com player, setlist e visualizadores de áudio reativos.

**Desenvolvido por:** [MagicPage](https://www.magicpage.com.br)

## Scripts

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Deploy

Projeto preparado para Vercel. Domínio de exemplo: `radioplay.tanolink.com.br`
